## Launch Strategy

### Timing

**Best days to launch:**
- Tuesday, Wednesday, or Thursday
- Avoid Fridays, weekends, and Mondays
- Launch at 12:01 AM PST (when PH day starts)

**Preparation timeline:**
- [ ] 2 weeks before: All assets ready
- [ ] 1 week before: Schedule launch date
- [ ] 3 days before: Notify supporters
- [ ] Day before: Final asset check
- [ ] Launch day: Monitor and respond to comments

### Pre-Launch Checklist

**Product Readiness:**
- [ ] App is deployed and accessible
- [ ] All features working on mobile
- [ ] No critical bugs
- [ ] Fast loading times
- [ ] Works on iOS Safari and Chrome
- [ ] PWA install flow tested

**Content Readiness:**
- [ ] Thumbnail (240x240px)
- [ ] Gallery images (3-5 images, 1270x760px)

## FAQs to Prepare

Have answers ready for:

1. **"How is this different from Strong/Hevy/Fitbod?"**
   - Focus on Iron Audit intelligence and pain-based progression

2. **"Is this free?"**
   - Clarify pricing model

3. **"Does it work offline?"**
   - Yes, it's a PWA with full offline support

4. **"Android/iOS support?"**
   - PWA works on both, explain installation process

5. **"Can I import my data from [other app]?"**
   - Current import capabilities

6. **"Is my data synced across devices?"**
   - Explain localStorage (device-only) approach

7. **"Will you add [feature X]?"**
   - Roadmap items and how to suggest features

## Competitor Research

Know these apps and how Schwarzy differs:
- **Strong**: More features, paid, no intelligent suggestions
- **Hevy**: Social features, but basic progression tracking
- **Fitbod**: AI workout generation, but doesn't track pain/form
- **JEFIT**: Comprehensive but cluttered UI

**Schwarzy's edge:** Pain-based intelligent progression + clean mobile UX + privacy-first
